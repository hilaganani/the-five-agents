# 🔍 לוג חיפושים של חן

זהו הזיכרון המתמשך שלי. **לפני** כל חיפוש אני בודקת כאן (Grep על מילות מפתח) אם כבר
חיפשתי משהו דומה ב-30 הימים האחרונים. **אחרי** כל חיפוש אני מוסיפה entry חדש.

פורמט ה-entry:

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
