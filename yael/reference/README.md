# תיקיית Reference — דוגמאות לסגנון הכתיבה שלנו

הניחו כאן קבצי `.md` עם טקסטים שמייצגים את הסגנון שלנו.

יעל תקרא את כל הקבצים בתיקייה זו בתחילת כל משימה כדי להבין את הסגנון לפני הכתיבה.

## כיצד להשתמש

- הוסיפו קבצים בשם `example-1.md`, `example-2.md` וכו'
- כל קובץ יכול להיות מאמר שלם, קטע, או אפילו כמה פסקאות לדוגמה
- ככל שיש יותר דוגמאות, כך יעל תבין טוב יותר את הסגנון
